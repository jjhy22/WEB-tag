HTML tage


html : head태그와 body태그를 감싸는 태그

head : 본문을 설명하는 태그

body : 본문을 나타내는 태그

title : 페이지의 이름을 나타내는 태그

a (anchor) : 태그의 제왕, 문서에 링크를 다는 태그
href (hypertext reference) : 링크의 주소 설명

img (image) : 사진 첨가 태그, 뒤에 어떤 사진인지 설명 필요(src=source)

p (paragraph) : 단란을 나타내는 태그

li (list) : 목록을 만드는 태그, ul,ol 태그 내부에 들어감

ul (unordered list) : li태그를 감싸는 태

br (line-break) : 줄바꿈 태그

h1~h6 (head) : 제목을 나타내는 태그, 숫자의 크기가 커질 수록 작아짐
